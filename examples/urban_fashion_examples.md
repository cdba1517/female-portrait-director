# 都市时尚写真示例

## 参数组合

```text
写真风格：都市时尚写真
场景方向：城市写字楼落地窗旁
服装方向：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质标签：清冷、利落、成熟自信
五官方向：高级厌世脸
身形方向：高挑模特比例
镜头方向：自然站姿，大腿以上构图
光线氛围：冷白自然光
滤镜效果：都市高级灰摄影滤镜
画幅比例：4:5
平台用途：社交媒体 / 公众号配图
补充要求：现代、简洁，不要廉价影楼感
```

## 模块解析

```text
1. 五官模块解析：高级厌世脸，脸部线条精致但不夸张，眼神冷淡克制，表情轻微疏离，适合都市杂志感画面。
2. 身形模块解析：高挑模特比例，肩颈舒展，腿部比例明显，服务于穿搭展示。
3. 服装模块解析：黑色西装外套结构利落，白色丝质内搭提供材质对比，高腰长裤拉长比例，整体完整清晰。
4. 场景模块解析：人物位于写字楼落地窗旁，前景保持简洁，中景突出人物，背景保留虚化城市建筑线条。
5. 镜头模块解析：自然站姿，大腿以上构图，突出穿搭层次、肩颈和高腰比例，避免僵硬摆拍。
6. 光线与滤镜模块解析：冷白自然光照亮五官和服装细节，高级都市街拍滤镜保持色彩干净、轻商业、适合传播。
```

## 最终提示词

```text
生成一张 4:5 竖版、真实摄影质感的都市女性时尚写真。画面主体是一位明确成年的东方女性，成年气质明确，整体清冷、利落、成熟自信。人物五官方向为高级厌世脸，脸部线条精致但不夸张，眼神冷淡克制，表情轻微疏离，妆容干净精致，不网红脸，不假面感。

人物身形为高挑模特比例，肩颈舒展，腿部比例明显，采用自然站姿和大腿以上构图，姿态平稳但不僵硬。人物穿着黑色西装外套、白色丝质内搭和高腰长裤，西装结构利落，丝质内搭形成克制的材质对比，高腰裤拉长整体比例，服装完整清晰。场景位于城市写字楼落地窗旁，前景保持简洁，中景突出人物主体，背景保留轻微虚化的现代建筑线条，画面秩序清晰，不使用杂乱街景。

光线为冷白自然光，照亮脸部和服装材质细节，阴影自然。使用高级都市街拍滤镜，色彩干净，人物与穿搭突出。整体呈现现代、成熟、有距离感的都市时尚摄影氛围。
```
