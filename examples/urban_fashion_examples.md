# 都市时尚写真示例

## 参数组合

```text
写真风格：都市时尚写真
场景方向：城市写字楼落地窗旁
服装方向：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质标签：清冷、利落、成熟自信
五官方向：高级厌世脸
身形方向：高挑模特比例
镜头方向：自然站姿，大腿以上构图
光线氛围：冷白自然光
滤镜效果：高级都市街拍滤镜
画幅比例：4:5
平台用途：公众号配图
```

## 导演式模块解析

选择会议间隙在落地窗前停步的短暂时刻。人物一手轻轻整理西装袖口，视线落向窗外城市线条。构图利用玻璃反光、建筑纵深和一侧留白建立秩序。

## 最终提示词

```text
生成一张 4:5 竖幅、真实摄影质感的都市女性时尚写真。画面中的人物是一位虚构、明确成年的东方女性。会议间隙，她在写字楼落地窗前短暂停步，一手轻轻整理黑色西装袖口，身体保持自然站姿，视线落向窗外。动作细小但利落，呈现成熟、自信而略带距离感的都市气质。

黑色西装外套结构清晰，白色丝质内搭在冷白自然光下形成克制的材质对比，高腰长裤拉长整体比例。大腿以上构图强调穿搭层次和肩颈线条。背景只保留玻璃反光与虚化建筑纵深，画面一侧预留适度文字空间。整体采用高级都市街拍滤镜，色彩干净，阴影自然，具有轻商业传播感。
```
