# 古风仙侠美人图示例

## 参数组合

```text
写真风格：古风仙侠美人图
场景方向：云雾山水间的古风庭院
服装方向：月白色飘逸古风长裙 + 银色细节
气质标签：清冷、疏离、仙气
五官方向：古典东方美人脸
身形方向：纤细清瘦身形
镜头方向：全身竖版构图
光线氛围：月白柔光
滤镜效果：古风清冷摄影滤镜
画幅比例：9:16
平台用途：古风角色图 / 社交媒体
补充要求：配色克制，保持清冷仙侠氛围
```

## 模块解析

```text
1. 五官模块解析：古典东方美人脸，脸型柔和古典，眉眼含蓄，眼型偏丹凤眼或细长杏眼，妆容精细克制。
2. 身形模块解析：纤细清瘦身形，肩颈线干净，四肢修长，姿态端庄稳定，适合古风全身构图。
3. 服装模块解析：月白色飘逸古风长裙为主色，冷灰为辅色，银色细节为点缀色。衣料轻盈，结构完整，避免现代化。
4. 场景模块解析：人物位于虚化古风庭院，前景可见轻微雾气，中景为人物，背景保留柔和柱廊线条、远山和薄雾。
5. 镜头模块解析：全身竖版构图，保持服装轮廓和人物仪态完整，避免头饰、服装和背景互相干扰。
6. 光线与滤镜模块解析：冷调柔光与清冷仙气古风滤镜营造冷白、柔雾、空灵气质，高光克制，发丝和银色细节有轻微边缘光。
```

## 最终提示词

```text
生成一张 9:16 竖版、具有真实摄影质感与东方古典氛围的古风仙侠女性人像。画面主体是一位明确成年的东方女性，成年气质明确，整体清冷、疏离、仙气克制。人物五官方向为古典东方美人脸，脸型柔和古典，眼型偏丹凤眼或细长杏眼，眉眼含蓄，眼神安静有距离感，妆容精细克制，不网红脸，不现代影楼感。

人物身形纤细清瘦，肩颈线干净，四肢修长，采用端庄自然的全身站姿。服装为月白色飘逸古风长裙，衣料轻盈，层次流动，结构完整；配色只使用月白主色、冷灰辅色和银色细节点缀，避免高饱和喜庆感和现代服装感。

场景位于云雾山水间的古风庭院，前景有轻微柔雾，中景为人物主体，背景可见弱化虚化的古典柱廊线条、远山和薄雾，具有层次但不过度繁复。光线为冷调柔光，银色细节和发丝带有轻微边缘光，使用清冷仙气古风滤镜，画面冷白、通透、安静、空灵，具有克制的仙侠氛围。
```
