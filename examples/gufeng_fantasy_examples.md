# 古风仙侠美人图示例

## 参数组合

```text
写真风格：古风仙侠美人图
场景方向：云雾山水间的古风庭院回廊
服装方向：月白色唐风幻想大袖衫 + 轻盈披帛 + 银色刺绣腰封
气质标签：清冷、疏离、仙气
五官方向：古典东方美人脸
身形方向：纤细清瘦身形
镜头方向：轻侧身站姿，半身到大腿构图
光线氛围：冷调柔光
滤镜效果：清冷仙气古风滤镜
画幅比例：9:16
平台用途：角色写真
```

## 导演式模块解析

选择人物穿过回廊后停步回望的一刻。披帛被薄风轻轻带起，身体转向幅度很小，眼神越过镜头落向远处。环境只保留柱廊线条、薄雾和淡远山水。

## 最终提示词

```text
生成一张 9:16 竖幅、真实摄影质感的古风仙侠女性人像。画面中的人物是一位虚构、明确成年的东方女性。她刚刚穿过云雾山水间的古风庭院回廊，在廊柱之间短暂停步，身体轻微转向，眼神越过镜头落向远处。薄风带起一侧披帛，画面安静而有细微流动感。

她拥有端庄含蓄的古典东方美人脸，妆容精细克制。月白色唐风幻想大袖衫形成轻盈层次，银色刺绣腰封收束轮廓，少量珍珠发饰提供细微光泽。背景只保留虚化柱廊、薄雾和淡远山水。冷调柔光落在脸侧、衣料褶皱和披帛边缘，形成轻微边缘光。整体采用清冷仙气古风滤镜，月白为主色，银色为辅色，珍珠光泽作为点缀。
```
