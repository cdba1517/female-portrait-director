# 电商服装模特图示例

## 参数组合

```text
写真风格：电商服装模特图
场景方向：干净摄影棚
服装方向：米白色针织连衣裙，完整展示版型、材质、领口、袖型、腰线和下摆
气质标签：自然商业模特感
五官方向：自然商业模特脸
身形方向：高挑模特比例
镜头方向：自然站姿，全身构图
光线氛围：棚拍柔光
滤镜效果：电商干净摄影滤镜
画幅比例：4:5
平台用途：电商主图
```

## 导演式模块解析

展示目标是完整呈现针织连衣裙。模特自然站立，手臂与身体保持轻微距离，不遮挡领口、腰线和下摆。摄影棚背景简洁，光线稳定覆盖纹理和轮廓。

## 最终提示词

```text
生成一张 4:5 竖幅、真实摄影质感的女性服装电商主图。画面中的人物是一位虚构、明确成年的东方女性模特，外形自然干净。她在浅灰摄影棚背景前自然站立，双臂与身体保持轻微距离，让服装结构完整可见。

米白色针织连衣裙是画面第一主体。准确呈现针织纹理、领口、袖型、肩线、腰线、下摆和整体版型，保持颜色真实，不增加遮挡性道具。棚拍柔光均匀覆盖衣料表面和轮廓，背景简洁，使用电商干净摄影滤镜。画面清晰稳定，适合作为电商主图。
```
