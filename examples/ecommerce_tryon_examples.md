# 电商服装模特图示例

## 参数组合

```text
写真风格：电商服装模特图
场景方向：干净摄影棚
服装方向：米白色针织连衣裙，完整展示版型、材质、领口、袖型、腰线和下摆
气质标签：自然商业模特感
五官方向：自然商业模特脸
身形方向：高挑模特比例
镜头方向：自然站姿，全身构图
光线氛围：摄影棚柔光
滤镜效果：电商干净摄影滤镜
画幅比例：4:5
平台用途：电商主图
补充要求：服装真实还原优先，避免遮挡和色差
```

## 模块解析

```text
1. 五官模块解析：自然商业模特脸，外形干净清秀，妆容自然，人物服务于服装展示，不喧宾夺主。
2. 身形模块解析：高挑模特比例，肩颈舒展，腿部比例明显，姿态稳定，适合完整展示服装。
3. 服装模块解析：米白色针织连衣裙是第一主体，必须准确呈现颜色、针织材质、领口、袖型、腰线、下摆和整体版型。
4. 场景模块解析：干净摄影棚，背景为浅灰、米白或纯白，前景和背景均不使用复杂道具。
5. 镜头模块解析：自然站姿，全身构图，服装完整可见，头发、手臂、道具和包袋不得遮挡重点。
6. 光线与滤镜模块解析：棚拍柔光稳定照亮服装纹理和轮廓，电商干净摄影滤镜确保颜色准确、画面清楚真实。
```

## 最终提示词

```text
生成一张 4:5 竖版、真实摄影质感的女性服装电商模特展示图。画面主体是一位明确成年的东方女性模特，成年气质明确，外形自然干净，五官方向为自然商业模特脸。人物只作为服装展示载体，不抢走服装主体。

模特必须穿着米白色针织连衣裙。服装是画面第一展示主体，需要完整、清晰、准确地展示米白色、针织材质、领口、袖型、腰线、下摆和整体版型，避免色差。模特身形为高挑模特比例，肩颈舒展，腿部比例自然，采用稳定自然的站姿和全身构图。不要让头发、手臂、包袋、道具或复杂姿势遮挡服装。

背景为干净摄影棚，使用浅灰、米白或纯白背景，不添加复杂道具。光线为棚拍柔光，稳定照亮服装纹理和轮廓，整体采用电商干净摄影滤镜。画面清晰、干净、真实，服装颜色准确，适合作为电商主图。
```
