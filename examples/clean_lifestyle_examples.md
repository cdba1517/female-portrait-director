# 清纯生活照示例

## 参数组合

```text
写真风格：清纯生活照
场景方向：咖啡馆靠窗座位
服装方向：白色针织开衫 + 浅色内搭
气质标签：温柔、自然、安静
画幅比例：9:16
平台用途：生活照
```

## 导演式模块解析

选择雨后午后的短暂阅读停顿。人物刚将书页轻轻合上，手指仍停在纸张边缘，视线越过玻璃看向窗外。白色针织开衫的袖口略微覆盖手腕，窗光落在脸侧和针织纹理上。环境只保留书页一角、玻璃上的淡淡雨痕和远处模糊街景。

## 最终提示词

```text
生成一张 9:16 竖幅、真实摄影质感的咖啡馆窗边生活照。画面中的人物是一位虚构、明确成年的东方女性。雨后的午后，她坐在临街窗边，刚把翻到一半的书轻轻合上，手指仍停留在书页边缘，视线越过玻璃看向窗外，没有刻意看镜头。这个短暂走神的瞬间让画面安静、温柔，像生活电影中没有对白的一帧。

她拥有柔和耐看的淡颜，眉眼舒展，妆容清透自然。白色针织开衫质地细腻柔软，搭配浅色内搭，袖口略微覆盖手腕。柔和窗光从侧前方落在脸侧、发丝和针织纹理上，玻璃上的淡淡雨痕与远处虚化街景形成轻微层次。前景只保留书页一角，背景不堆叠复杂道具。整体使用奶油暖白生活剧照滤镜，保留真实肤质、柔和阴影和自然日常感。
```
