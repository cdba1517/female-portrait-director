# 清纯生活照示例

## 参数组合

```text
写真风格：清纯生活照
场景方向：咖啡馆靠窗座位
服装方向：白色针织开衫 + 浅色内搭
气质标签：清纯温柔、自然亲近
五官方向：初恋感淡颜
身形方向：自然协调身形
镜头方向：自然坐姿，大腿以上构图
光线氛围：柔和自然光
滤镜效果：奶油暖白生活剧照滤镜
画幅比例：9:16
平台用途：生活照 / 小红书种草
补充要求：真实自然，不要过度摆拍
```

## 模块解析

```text
1. 五官模块解析：初恋感淡颜，脸型柔和流畅，眉眼舒展，眼神干净温柔，鼻型秀气，唇形自然柔软，妆容清透淡雅。
2. 身形模块解析：自然协调身形，肩颈、腰线和腿部比例自然，不刻意强化曲线；坐姿放松，适合生活照。
3. 服装模块解析：白色或米白色针织开衫材质柔软细腻，浅色基础内搭干净克制，适合咖啡馆生活场景。
4. 场景模块解析：人物坐在咖啡馆靠窗座位，前景有浅色桌面和咖啡杯，中景为人物，背景保留柔和窗光、木质家具和虚化绿植。
5. 镜头模块解析：自然坐姿，大腿以上构图，兼顾五官、上衣层次和生活氛围。
6. 光线与滤镜模块解析：柔和自然光照亮脸部和服装纹理，奶油暖白滤镜让肤色自然通透，画面像生活电影中的安静片段。
```
