# 导演式扩写规范

## 核心原则

扩写不是补齐字段。扩写要创造一个真实摄影中可能发生的瞬间。

先确定画面意图，再选择细节。不要把场景库中的道具全部塞进提示词。

## 必须建立的五个要素

1. **时间切片**：午后、雨后、傍晚、清晨、室内灯刚亮起等。
2. **轻微事件**：刚合上书、端起咖啡前停顿、海风带动开衫、在回廊转身停步等。
3. **动作链**：手部动作、身体重心、肩颈状态和衣料动态要彼此合理。
4. **视线落点**：看向窗外、低头看书页、略过镜头、望向远处海面等。
5. **选择性细节**：只保留两到三个最能支持氛围的环境元素。

## 组织方式

最终提示词应自然融合以下内容，不必固定分段：

- 主体：成年边界、气质、脸部和妆容。
- 瞬间：人物正在做什么，为什么姿态自然。
- 服装：材质、版型、层次和衣料动态。
- 空间：人物所在位置、前中后景关系和留白。
- 镜头：景别、机位、人物在画面中的关系。
- 光线：具体落点、阴影过渡、肤质和衣料表现。
- 滤镜：色彩、饱和度、颗粒或清透感。

## 选择规则

- 每张图只使用一个主事件。
- 每张图只保留两到三个环境细节。
- 不要在所有咖啡馆画面中固定加入桌面、咖啡杯和绿植。
- 不要在所有海边画面中固定加入栏杆、海面和天空渐变。
- 不要把负面约束重复塞进最终正向提示词；正向提示词优先描述想要的画面。
- 示例只用于理解完成度，不用于复制句式。

## 平台用途转换

| 平台用途 | 构图处理 |
| --- | --- |
| 生活照 / 社交媒体 | 保留环境呼吸感和自然瞬间，避免过度商业摆拍 |
| 小红书种草 | 保留穿搭层次，主体清晰，可预留少量标题空间 |
| 公众号配图 | 构图稳定，适度留白，降低背景干扰 |
| 电商主图 | 商品完整清晰，减少氛围道具，不遮挡重点 |
| 电商详情页 | 可加强材质、局部结构和版型说明 |

## 快速自检

如果删掉字段名称后，提示词仍像一张可以想象出来的照片，扩写才算完成。
