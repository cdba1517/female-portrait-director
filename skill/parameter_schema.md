# 参数字段

锁定用户明确填写的字段，只对缺失字段执行合理补全。字段补全不得替换用户方向。

| 字段 | 说明 | 默认值 |
| --- | --- | --- |
| 写真风格 | 选择唯一主路由 | 根据用户意图判断；无法判断时使用清纯生活照 |
| 场景方向 | 指定拍摄环境 | 与主路由协调的简洁场景 |
| 服装方向 | 描述颜色、版型、材质和搭配 | 与主路由匹配的完整服装 |
| 配色方向 | 控制主色、辅色、金属或珠宝点缀色及其占比 | 与主路由协调；用户指定时锁定 |
| 华丽度 | 控制刺绣、材质、发饰、首饰、服装层次和背景复杂度 | 与主路由协调；不适用时不特别强调 |
| 年龄特征 | 控制可见年龄段与成年表现 | 24–28 岁年轻成年东方女性，成年气质明确 |
| 气质标签 | 描述人物整体气质 | 自然、干净、明确成年 |
| 五官方向 | 控制脸型、眉眼、神情和妆容 | 与主路由协调的东方女性五官 |
| 身形方向 | 控制比例、姿态和镜头配合 | 自然协调身形 |
| 身形吸引力强度 | 控制线条表达程度 | 弱或中 |
| 线条重点 | 指定需要通过姿态、服装和光线呈现的区域 | 与服装和镜头匹配 |
| 镜头方向 | 控制景别、姿态和重点 | 半身到大腿或大腿以上 |
| 光线氛围 | 控制光源效果 | 柔和自然光 |
| 滤镜效果 | 控制色彩和摄影质感 | 与主路由匹配 |
| 画幅比例 | 控制成图比例 | `9:16` |
| 尺寸参数 | 控制像素尺寸、分辨率或宽高；仅在用户明确填写时锁定，不得虚构 | 未指定 |
| 平台用途 | 控制构图和信息密度 | 生活照 |
| 输出模式 | 控制返回内容 | 标准：参数锁定 + 逐段导演式模块扩写 + 五段式最终提示词 + 负面约束；最终提示词与负面约束分别使用 Markdown `text` 代码块 |
| 补充要求 | 用户额外限制 | 保持真实、完整、风格统一 |

## 支持风格

- 清纯生活照
- 纯欲曲线生活照
- 都市时尚写真
- 古风仙侠美人图
- 电商服装模特图
- 复古港风写真
- 法式慵懒写真
- 新中式东方写真
- 活力运动写真
- 旅行假日写真
- 影楼精修写真
- 东方丰腴写真
- 清冷仙气古风增强版
- 明媚华贵古风增强版

以上为当前已经实现的 14 条 Route。更多扩展方向可在 [style-registry.md](style-registry.md) 中预注册，但对应文件未创建前不得虚构 Route 内容。

## 输出模式

- `简洁`
- `标准`
- `导演详版`
- `直接生成图片`

## 别名规范化

别名只用于内部规范化，不需要提示用户修改输入。

| 输入别名 | 规范名称 |
| --- | --- |
| 摄影棚柔光 | 棚拍柔光 |
| 月白柔光 | 冷调柔光 |
| 古风清冷摄影滤镜 | 清冷仙气古风滤镜 |
| 都市高级灰摄影滤镜 | 高级都市街拍滤镜 |
| 干净清秀脸 | 邻家清秀脸 |

`自然商业模特脸` 是电商路线的正式五官选项。
