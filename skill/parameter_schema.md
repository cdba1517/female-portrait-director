# 参数字段

系统必须锁定用户明确填写的字段，只对缺失字段执行合理补全。对已锁定参数执行扩写、细化、稳定化和视觉化解释，不得替换用户方向。

| 字段 | 说明 | 推荐选项 | 默认值 |
| --- | --- | --- | --- |
| 写真风格 | 选择主风格路由 | 清纯生活照、纯欲曲线生活照、都市时尚写真、古风仙侠美人图、电商服装模特图 | 根据用户意图判断；无法判断时使用清纯生活照 |
| 场景方向 | 指定拍摄环境 | 咖啡馆、海边步道、城市街头、写字楼、摄影棚、古风庭院、山水云雾场景 | 与主风格协调的简洁场景 |
| 服装方向 | 描述颜色、版型、材质和搭配 | 生活穿搭、时尚通勤、古风服饰、电商待展示服装 | 与主风格匹配的完整服装 |
| 气质标签 | 描述人物整体气质 | 清纯温柔、安静克制、清冷利落、成熟自信、疏离仙气、自然商业模特感 | 自然、干净、明确成年 |
| 五官方向 | 控制脸型、眉眼、眼神、鼻型、唇形和妆容 | 初恋感淡颜、清冷淡颜、邻家清秀脸、精致东方淡颜、明艳东方浓颜、甜美元气脸、高级厌世脸、复古港风脸、古典东方美人脸 | 与主风格协调的东方女性五官 |
| 身形方向 | 控制身体比例、线条、姿态和镜头配合 | 自然协调身形、纤细清瘦身形、自然吸引力曲线、胸腰腿线条感、丰腴曲线、健康运动线条、高挑模特比例 | 自然协调身形 |
| 身形吸引力强度 | 控制线条表达强度 | 弱、中、强；始终保持成年、时尚、克制、服装完整 | 弱或中 |
| 线条重点 | 指定适度强调区域，并转译到服装、姿态、镜头和光线 | 肩颈、锁骨、腰线、小腹、腿部比例、整体轮廓 | 与服装和镜头匹配 |
| 镜头方向 | 控制景别、姿态和重点展示部位 | 半身近景、半身到大腿、大腿以上、全身、轻侧身站姿、自然坐姿、回头看镜头、倚靠姿态、对镜自拍 | 半身到大腿或大腿以上 |
| 光线氛围 | 控制光源如何作用于人物和环境 | 柔和自然光、柔和冷白自然光、奶油暖白自然光、傍晚柔和自然光、冷调柔光、棚拍柔光 | 柔和自然光 |
| 滤镜效果 | 控制画面色彩、肌肤质感和摄影气氛 | 奶油暖白生活剧照、冷白纯欲生活照、真实电影生活剧照、高级都市街拍、清冷仙气古风、电商干净摄影 | 与主风格匹配 |
| 画幅比例 | 控制成图比例 | 9:16、4:5、3:4、1:1、16:9 | 9:16 |
| 平台用途 | 控制构图目标 | 生活照、小红书种草、社交媒体、公众号配图、电商主图、电商详情页 | 根据场景判断 |
| 补充要求 | 用户的额外限制 | 构图、背景、姿态、颜色、展示重点等 | 保持真实、克制、完整服装 |

## 模块扩写规则

系统不得只复述参数名称。每次生成时需要将参数展开为：

```text
五官：脸型 + 眉眼 + 眼神 + 鼻型 + 唇形 + 妆容 + 气质 + 避免项
身形：身体比例 + 线条重点 + 姿态 + 镜头配合 + 服装配合 + 安全边界
服装：颜色 + 材质 + 版型 + 层次 + 长度 + 场景关系 + 完整性
场景：空间类型 + 人物位置 + 前景 + 中景 + 背景 + 虚化程度 + 复杂度
镜头：构图范围 + 姿态 + 身体关系 + 展示重点 + 稳定性
光线与滤镜：光线落点 + 阴影 + 色调 + 肌肤质感 + 整体氛围
```
