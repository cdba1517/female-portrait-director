# 公开调用说明

这个 Skill 会把少量女性人像参数扩写成完整、稳定、可直接复制的 AI 生图提示词。它会先完成内部导演设计，再锁定并扩写年龄特征、五官、服装、身体线条、姿态、场景、镜头、光线和滤镜，最后把这些模块融合为同一个自然瞬间。

## 输出模式

- `简洁`：只返回最终提示词和负面约束。
- `标准`：默认模式。返回参数锁定、逐段导演式模块扩写、五段式最终提示词和负面约束。模块扩写至少包含年龄与五官、身形与仪态、服装配色与发饰、华丽度的可见设计、场景、镜头、光线与滤镜。最终提示词和负面约束分别使用独立 Markdown `text` 代码块，便于直接复制。
- `导演详版`：在标准模式基础上，进一步说明瞬间、动作链、选择性环境细节、参数扩写关系、镜头目的与光线设计。
- `直接生成图片`：内部扩写后直接调用可用的图片生成能力。

直接说 `只要最终提示词` 等同于选择 `简洁`。

最终提示词第一句必须写明画幅比例。用户额外填写像素尺寸、分辨率或宽高时，系统必须保留原值并一并写入第一句；用户未填写时不得虚构具体像素值。

## 使用边界

默认人物必须是虚构、明确成年的女性。完整边界见 [../docs/prompt_safety.md](../docs/prompt_safety.md)。
