# 公开调用说明

这个 Skill 会把少量女性人像参数扩写成完整、稳定、可直接复制的 AI 生图提示词。它会锁定用户参数，再像摄影导演一样补充自然瞬间、动作链、视线、空间关系、镜头和光线。

## 快速调用

```text
风格：
场景：
服装：
气质：
画幅：
```

```text
输出模式：

```

## 支持风格

1. 清纯生活照
2. 纯欲曲线生活照
3. 都市时尚写真
4. 古风仙侠美人图
5. 电商服装模特图

## 输出模式

- `简洁`：只返回最终提示词和负面约束。
- `标准`：返回参数锁定、导演式模块解析、最终提示词和负面约束。
- `导演详版`：详细说明瞬间、动作链、镜头与光线设计。
- `直接生成图片`：内部扩写后直接调用可用的图片生成能力。

直接说 `只要最终提示词` 等同于选择 `简洁`。

## 使用边界

默认人物必须是虚构、明确成年的女性。完整边界见 [../docs/prompt_safety.md](../docs/prompt_safety.md)。
