# 公开调用说明

## Skill 是什么

女性人像提示词导演 Skill 用于将少量创作要求扩写为稳定、完整、可直接复制的女性人像 AI 生图提示词。系统会锁定用户参数，理解背后的视觉意图，再对五官、身形、服装、场景、镜头、光线、滤镜和负面约束进行摄影导演式扩写。

## 如何输入参数

最简调用只需提供以下字段：

```text
风格：
场景：
服装：
气质：
画幅：
```

需要更精细控制时，可以使用 [parameter_schema.md](parameter_schema.md) 中的完整字段。

## 支持风格

1. 清纯生活照
2. 纯欲曲线生活照
3. 都市时尚写真
4. 古风仙侠美人图
5. 电商服装模特图

## 示例调用

```text
风格：都市时尚写真
场景：城市写字楼落地窗旁
服装：黑色西装外套 + 白色丝质内搭 + 高腰长裤
气质：清冷、利落、成熟
镜头：大腿以上构图
光线：冷白自然光
画幅：4:5
```

## 输出格式

```text
一、参数锁定结果
二、模块解析
三、最终提示词
四、负面约束
```

模块解析包括：

```text
1. 五官模块解析
2. 身形模块解析
3. 服装模块解析
4. 场景模块解析
5. 镜头模块解析
6. 光线与滤镜模块解析
```

如果只需要可复制提示词，可以明确说“只要最终提示词”。系统将只输出最终提示词和负面约束。

## 使用边界

- 默认人物必须是虚构、明确成年的女性。
- 仅生成非裸露、非色情、非伤害性的视觉提示词。
- 不得用于未成年人性化、年龄模糊人物性化、非自愿图像、欺骗性身份内容或侵犯隐私的用途。
- 使用者需要遵守目标 AI 平台政策和所在地区法律法规。
