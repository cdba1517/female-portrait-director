# 风格指南

五大风格只选择一个主路由。用户参数发生冲突时，保留用户明确要求，通过背景简化、颜色克制、光线调整和构图补充降低冲突。

## V1.1 导演式扩写流程

每次生成时，不要把参数机械填入固定句式。先拆解五官、身形、服装、场景、镜头姿态、光线和滤镜模块，再将它们融合为自然、完整、可直接复制的视觉描述。

- 五官：扩写脸型、眉眼、眼神、鼻型、唇形、妆容和需要避免的问题。
- 身形：将比例、线条重点、姿态、镜头和服装联动处理。
- 服装：解析颜色、材质、版型、层次、长度和场景适配度。
- 场景：补充前景、中景、背景、虚化程度和环境复杂度。
- 镜头：明确构图范围、人物姿态、重点展示部位和稳定性。
- 光线与滤镜：说明它们如何影响肤色、服装细节、阴影和整体气氛。

## 关键优先级

- 清纯生活照：自然生活感优先。
- 纯欲曲线生活照：成年、安全、克制优先。
- 都市时尚写真：成熟气质和画面秩序优先。
- 古风仙侠美人图：风格统一和配色协调优先。
- 电商服装模特图：服装真实还原 > 服装完整清晰 > 模特好看 > 背景氛围。

## 风格隔离

- 清纯生活照不得自动强化曲线和过强性感表达。
- 纯欲曲线生活照不得变成低俗性感写真或情趣风。
- 古风仙侠不得混入现代吊带、现代街拍和现代妆造。
- 清冷仙气古风不得混入甜暖、喜庆或过艳氛围。
- 电商服装图不得为追求人像氛围牺牲服装清晰度。
- 都市时尚写真不得使用杂乱背景或模糊主体。
