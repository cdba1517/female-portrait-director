## 01｜五个参数，生成一张不像摆拍的咖啡馆生活照

成片：`../assets/cases/01-clean-lifestyle.png`

### 封面标题

```text
为什么你的 AI 生活照总像影楼摆拍？
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：清纯生活照
场景：午后咖啡馆靠窗座位
服装：米白针织开衫 + 浅色内搭
气质：温柔、自然、明确成年
画幅：3:4
```

### 发布正文

很多 AI 生活照的问题不是“不够漂亮”，而是人物、动作、光线和场景彼此没有关系，看起来像把几个标签拼在一起。

这个案例只给了五个参数。Skill 补全的是一个能被拍下来的瞬间：她刚把咖啡杯放回桌面，身体轻轻转向镜头，窗边光落在脸侧和针织纹理上，背景只保留玻璃窗、浅色桌面和咖啡杯。

它没有继续堆“高级感、氛围感、电影感”，而是先解决人物为什么在这里、手在做什么、光从哪里来。

项目与安装命令见评论区 / 文末。

### 推荐标签

`#Codex #AgentSkills #AI写真 #AI生图 #提示词`

## 02｜一张都市街拍，不只是让人物站在街上

成片：`../assets/cases/02-urban-fashion.png`

### 封面标题

```text
都市街拍的“高级感”，具体来自哪里？
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：都市时尚写真
场景：现代城市玻璃橱窗前
服装：炭灰西装 + 米白内搭 + 高腰阔腿裤
气质：轻熟、自信、克制
画幅：3:4
```

### 发布正文

“都市、时尚、高级、街拍”通常只会得到一张普通站姿照片。

这次 Skill 把它改成一个街拍动作链：人物在行走中短暂停下，重心稳定，外套自然垂落，玻璃橱窗提供侧前方反射光，建筑线条和城市倒影只做背景层次。

真正影响结果的不是形容词数量，而是穿搭廓形、人物重心、镜头距离、光线落点和背景秩序是否统一。

### 推荐标签

`#AI摄影 #OOTD #街拍 #Codex #PromptEngineering`

## 03｜复杂古风造型，怎样避免变成廉价 cosplay

成片：`../assets/cases/03-gufeng-xianxia.png`

### 封面标题

```text
古风图最容易翻车的，不是脸，而是服装结构
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：古风仙侠美人图
场景：云雾山水庭院回廊
服装：月白唐风大袖衫 + 冰蓝披帛 + 银色刺绣腰封
气质：清雅、梦幻、仙气、明确成年
画幅：3:4
```

### 发布正文

古风提示词经常同时塞入华服、披帛、头饰、宫殿、山水、云雾和各种光效，最后每个元素都有一点，但没有一个结构是清楚的。

这个案例把配色限制为月白主色、冰蓝辅色、银色细节点缀；背景只保留回廊、远山和薄雾；人物只做“转身停步、轻扶披帛”这一个事件。

复杂造型的关键不是继续增加元素，而是先决定哪些细节必须被看见。

### 推荐标签

`#古风 #仙侠 #AI绘画 #角色设计 #AgentSkills`

## 04｜电商模特图：人物好看，但服装必须更重要

成片：`../assets/cases/04-ecommerce-model.png`

### 封面标题

```text
为什么很多 AI 电商图最后变成了普通写真？
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：电商服装模特图
商品：象牙白短款结构外套，黑色滚边、金属纽扣、长袖
搭配：同色高腰阔腿裤
背景：浅灰摄影棚
要求：服装完整、颜色准确、无遮挡
画幅：3:4
```

### 发布正文

电商图和普通写真最大的区别，是优先级。

这个案例里，人物只是服装展示载体。镜头需要看清领口、肩线、袖长、纽扣、下摆、腰线和裤装比例；手臂、头发和道具都不能遮挡商品；光线也要优先保证颜色和面料准确。

如果提示词一直强调人物气质和氛围，却没有明确商品结构，模型自然会把它生成普通写真。

### 推荐标签

`#AI电商 #服装设计 #虚拟模特 #商品图 #AI工作流`

## 05｜港风不是“红唇 + 霓虹”两个标签

成片：`../assets/cases/05-retro-hongkong.png`

### 封面标题

```text
怎样让港风像电影，而不是夜店滤镜？
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：复古港风写真
场景：雨后旧街道、茶餐厅门口
服装：红色针织上衣 + 深色半裙
气质：明艳、有故事感、明确成年
画幅：3:4
```

### 发布正文

只写“港风、红唇、卷发、霓虹”，很容易得到高饱和夜店感。

这个案例把霓虹降为背景轮廓光，让脸部由暖调胶片光照亮；场景只留湿润路面、玻璃窗和虚化灯牌；人物刚在茶餐厅门口停下，轻轻回头看向镜头。

风格不是滤镜名称，而是人物、事件、场景和光线共同形成的摄影逻辑。

### 推荐标签

`#港风 #胶片摄影 #AI写真 #电影感 #Codex`

## 06｜新中式最重要的不是堆中式元素，而是留白

成片：`../assets/cases/06-new-chinese.png`

### 封面标题

```text
新中式人像：少放几个元素，画面反而更东方
```

### 最小输入

```text
使用 $female-portrait-director 直接生成图片：
风格：新中式东方写真
场景：安静茶室、屏风、竹影
服装：宣纸白盘扣上衣 + 低饱和玉绿色棉麻长裙
气质：清雅、知性、克制、明确成年
画幅：3:4
```

### 发布正文

新中式画面最常见的问题，是把屏风、竹子、香炉、茶具、书法和红木家具全部塞进背景。

这个案例只保留木质桌面、低饱和屏风和一小片竹影。人物刚把茶杯放回桌面，柔和侧光照亮脸侧、盘扣和棉麻纹理，其余空间保持安静留白。

东方感不一定来自更多符号，也可以来自姿态、材质、光影和空间秩序。

### 推荐标签

`#新中式 #东方美学 #AI摄影 #女性人像 #AgentSkills`
