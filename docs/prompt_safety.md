# 提示词安全边界

## 基本要求

- 人物必须是虚构、明确成年的女性。
- 描述成年气质，避免幼态化、少女儿童化、年龄模糊或未成年感表达。
- 服装必须完整，避免裸露、色情、情趣化和低俗擦边表达。
- 姿态自然克制，避免挑逗、强性暗示和非自愿情境。
- 不生成真实人物、明星或可识别个人的欺骗性身份图像。
- 不生成侵犯隐私、骚扰、诽谤或违法违规用途的内容。

## 安全表达建议

| 避免使用 | 推荐表达 |
| --- | --- |
| 性感 | 有吸引力、女性线条美、时尚写真感 |
| 诱惑 | 克制吸引力、情绪张力 |
| 暴露 | 轻盈穿搭、露肤克制、服装完整 |
| 丰满胸部 | 自然胸腰比例、身形曲线协调 |
| 挑逗 | 自然姿态、眼神克制 |
| 情趣 | 时尚内搭、生活化穿搭、高级私服 |

## 纯欲曲线风格

纯欲、曲线和吸引力只能表达为明确成年女性的时尚写真、身体线条美、克制吸引力和真实摄影美感。不得转化为裸露、色情、情趣化、挑逗姿态或年龄模糊表达。

## 审查友好改写

改写时保留原风格、场景、服装、构图、光线、滤镜和气质目标，同时强化：

```text
年轻成年女性
成年气质明确
服装完整
非裸露
非色情
非低俗
非情趣化
非未成年感
真实摄影质感
克制表达
```

## 输出前安全自检

在输出最终提示词前检查：

1. 人物是否明确成年，是否避免幼态化和年龄模糊表达。
2. 服装是否完整自然，是否避免裸露、情色、情趣化和低俗擦边。
3. 身形线条是否通过服装、姿态、镜头和光线自然体现，而不是夸张堆砌。
4. 姿态是否自然稳定，是否避免挑逗动作和强性暗示。
5. 最终提示词是否保留用户参数，同时满足目标平台政策和适用法律法规。
