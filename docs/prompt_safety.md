# 提示词安全边界

本文件是安全规则的唯一权威来源。其他文件只引用本文件，不重复维护完整规则。

## 基本要求

- 人物必须是虚构、明确成年的女性。
- 避免幼态化、儿童化、年龄模糊或未成年感表达。
- 服装必须完整，避免裸露、色情、情趣化和低俗擦边表达。
- 姿态自然，避免挑逗、强性暗示和非自愿情境。
- 不生成真实人物、明星或可识别个人的欺骗性身份图像。
- 不生成侵犯隐私、骚扰、诽谤或违法违规用途的内容。

## 纯欲曲线风格

纯欲、曲线和吸引力只能表达为明确成年女性的时尚写真、自然身体线条、真实摄影美感和有控制力的吸引力。不得转化为裸露、色情、情趣化、挑逗姿态或年龄模糊表达。

## 输出前检查

1. 人物是否明确成年。
2. 服装是否完整自然。
3. 身形线条是否通过服装、姿态、镜头和光线自然体现。
4. 姿态是否稳定自然。
5. 是否避免欺骗性身份内容和侵犯隐私用途。
6. 是否符合目标平台政策和适用法律法规。
